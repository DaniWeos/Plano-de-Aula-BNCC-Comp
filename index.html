Gerador de Plano de Aula
⚡ Preencher via JSON

Como gerar o JSON da sua aula:

  1. Grave um áudio descrevendo a aula (use o gravador do celular ou qualquer app de voz).
  2. Abra uma IA — ChatGPT, Gemini ou Copilot — e envie o áudio ou a transcrição.
  3. Use o comando abaixo para pedir o JSON:
“Com base neste áudio/texto, gere um JSON com os seguintes campos para um plano de aula: instituicao, disciplina, habilidade, tema, assunto, data, tempo, objetivos, conhecimentos, conteudo, metodologia, recursos, avaliacao, referencias. Para campos de lista, separe cada item com . Retorne apenas o JSON puro, sem explicações.”

4. Copie o JSON gerado, cole no campo abaixo e clique em Preencher campos.
